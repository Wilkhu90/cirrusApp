About Cirrus App

What is Palindrome?

A palindrome is a word, phrase, number, or other sequence of characters which reads the same backward as forward, such as madam or racecar. - Wikipedia
A palindromic number or numeral palindrome is a number that remains the same when its digits are reversed. Like 16461. - Wikipedia

Features of the Application

1Compute the largest palindrome in the given range of numbers.
2View all the palindromes in the given range of numbers.
3Automatically stores and retrieves the previously computed palindromes from local storage.

How to use it?

Click to run the app.

1Choose values for lower and upper bound.
2Choose if you want to see all palindromes in the selected range.(Optional).
3Submit and you're done. Enjoy the App!!.
